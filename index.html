信导期中考题解

发表于 2024-11-15 更新于 2024-11-15 分类于 大学

考试环境

  • 操作系统: Windows10
  • 终端 cmd, 古董 powershell
  • 编辑器: vscode (包含插件 pylance, python debugger, pylint), vim (版本 \(9.1\)), gvim
  • IDE: pycharm
  • 外设: 键盘还可以, 鼠标手感稀烂 (有的机房可能键盘也烂)

考试题目

第一场

A

题意: 输入 \(x\), \(n\), 输出 \(x^n + 1\)

B

题意: 输入数列, 输出中位数.

C

题意: 输入字符串, 偶数下标转大写, 奇数下标转小写. 最后逆序输出.

D

题意: 实现一个类, 完成其中的三个函数 (具体变量名点忘了). 初始化类的时候传入两个 str; __str__ 函数需要格式化字符串; 第三个函数传入 list[str], 统计其中每个元素出现的次数.

第二场

本人没有参加第二场考试, 无法给出实现代码. 简要题意如下:

  • A: 输入 \(n\), 输出 \(1 + 2 + \cdots + n\);
  • B: 输入数列, 输出第 \(k\) 大数;
  • C: 输入字符串, 把其中连续部分压缩. (可能是 aaa -> 3a 这种?)
  • D: 也是实现一个类, 完成几个任务.

考试总结

评测机环境比想象中好, 不过还是建议装个新版本 pwsh.

考试时间 \(45\) 分钟, 我花了快 \(30\) 分钟, 时间应该不是特别紧张 (?).

感觉题目还可以, 考察了简单的输入输出, 数组切片 (逆序输出), 常用函数 len(), sorted(), 字符串相关的大小写转换, 类初始化与魔术方法. 个人建议第一题数学公式可以复杂一点.
